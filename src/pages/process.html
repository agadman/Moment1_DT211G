Arbetsprocess

Automatisering och Publicering

Uppgiftsbeskrivning

Den här uppgiften ingår i kursen Frontend-baserad webbutveckling. Jag ska använda Node.js för att automatisera och förenkla webbutveckling, skapa en effektiv utvecklingsmiljö och göra automatisk publicering möjlig. Fokus är att bygga en grundstruktur (boilerplate) för framtida projekt.

Syfte och mål

Målet med uppgiften är att:

  • Lära mig automatisera uppgifter som minifiering av HTML, CSS och JavaScript.
  • Optimera bilder (WebP/AVIF).
  • Säkerställa bakåtkompatibilitet med transpilering av JavaScript.
  • Skapa en boilerplate som kan återanvändas i andra projekt.
  • Implementera automatisk publicering via Git och hostar som Netlify, Vercel eller GitHub Pages.

Boilerplate

En boilerplate är en färdig kodbas med grundläggande struktur och verktyg för att snabbt starta ett projekt. Min boilerplate kommer att inkludera:

  • Strukturerad mappning för HTML, CSS, JavaScript och bilder.
  • Konfiguration för att automatisera minifiering, transpilering och bildoptimering.
